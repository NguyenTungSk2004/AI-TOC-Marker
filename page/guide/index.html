Tải về và giải nén AI TOC Marker

Bước đầu tiên là tải file ZIP từ GitHub và giải nén để chuẩn bị cài đặt

Chi tiết thực hiện:

  1. Truy cập trang GitHub Releases
  2. Tải file AI-TOC-Marker-x.x.x.zip về máy
  3. Click chuột phải vào file ZIP → chọn "Giải nén tất cả..."
  4. Chọn thư mục đích và nhấn "Giải nén"
Mẹo: Nên giải nén vào thư mục dễ tìm như Desktop hoặc Documents
File ZIP sau khi tải về và quá trình giải nén
2

Mở Google Chrome

Khởi động trình duyệt Google Chrome để bắt đầu quá trình cài đặt tiện ích

Chi tiết thực hiện:

  1. Tìm và nhấp đúp vào biểu tượng Google Chrome trên Desktop
  2. Hoặc nhấn tổ hợp phím Windows + R → gõ chrome → Enter
  3. Đợi Chrome khởi động hoàn tất
Lưu ý: Phiên bản Chrome cần từ 88 trở lên để hỗ trợ tốt nhất
Giao diện Google Chrome sau khi khởi động
3

Truy cập trang quản lý tiện ích

Mở trang Chrome Extensions để có thể cài đặt tiện ích từ file local

Chi tiết thực hiện:

  1. Nhấp vào thanh địa chỉ (address bar) của Chrome
  2. Gõ chính xác: chrome://extensions
  3. Nhấn phím Enter để truy cập
  4. Trang Extensions sẽ mở ra với danh sách các tiện ích đã cài
Cách khác: Menu ⋮ → Extensions → Manage Extensions
Trang quản lý Extensions của Chrome
4

Bật chế độ Developer Mode

Kích hoạt Developer Mode để có thể cài đặt tiện ích từ file thay vì Chrome Store

Chi tiết thực hiện:

  1. Tìm công tắc "Developer mode" ở góc phải trên cùng
  2. Nhấp vào công tắc để chuyển từ OFF sang ON
  3. Công tắc sẽ chuyển sang màu xanh và hiện thêm 3 nút mới
  4. Kiểm tra xem đã có các nút: Load unpacked, Pack extension, Update
Bảo mật: Developer Mode cho phép cài tiện ích từ nguồn không chính thức, chỉ bật khi cần thiết
Công tắc Developer mode đã được bật (màu xanh)
5

Load Unpacked Extension

Chọn thư mục đã giải nén để Chrome cài đặt AI TOC Marker vào hệ thống

Chi tiết thực hiện:

  1. Nhấn nút "Load unpacked" (màu xanh) vừa xuất hiện
  2. Hộp thoại chọn folder sẽ mở ra
  3. Điều hướng đến thư mục đã giải nén ở Bước 1
  4. Chọn thư mục chứa file manifest.json → nhấn "Select Folder"
  5. Chrome sẽ tự động cài đặt và hiển thị AI TOC Marker trong danh sách
Dấu hiệu thành công: Tiện ích xuất hiện với logo AI TOC Marker và trạng thái "Enabled"
Chọn thư mục chứa file manifest.json của AI TOC Marker
6

Ghim tiện ích lên thanh công cụ

Pin AI TOC Marker vào toolbar để dễ dàng truy cập khi sử dụng ChatGPT

Chi tiết thực hiện:

  1. Nhấp vào biểu tượng puzzle (🧩) trên thanh công cụ Chrome
  2. Tìm "AI TOC Marker" trong danh sách extensions
  3. Nhấp vào biểu tượng pin (📌) bên cạnh tên tiện ích
  4. Icon AI TOC Marker sẽ xuất hiện trực tiếp trên thanh công cụ
Khuyên dùng: Ghim giúp truy cập nhanh mà không cần mở menu extensions mỗi lần
Ghim tiện ích để truy cập nhanh từ thanh công cụ
7

Hoàn tất! Sử dụng trên ChatGPT

Bây giờ bạn có thể trải nghiệm AI TOC Marker trên ChatGPT để tạo mục lục tự động

Chi tiết thực hiện:

  1. Mở tab mới và truy cập ChatGPT
  2. Đăng nhập vào tài khoản ChatGPT của bạn
  3. Bắt đầu cuộc hội thoại có nội dung dài với nhiều phần
  4. Nhấp vào icon AI TOC Marker trên thanh công cụ
  5. Tiện ích sẽ tự động tạo mục lục cho cuộc hội thoại
Chúc mừng! Bạn đã cài đặt thành công AI TOC Marker. Giờ đây ChatGPT sẽ thân thiện hơn bao giờ hết!
